# Блогикум

## О проекте

Блогикум — это платформа для ведения личных блогов. Пользователи могут создавать публикации, добавлять к ним изображения, оставлять комментарии.

![Главная страница](кк.png)

## Используемые технологии

- Python 3.9+
- Django 4.2
- SQLite
- Bootstrap 5
- Pillow

## Возможности

- Регистрация и авторизация
- Создание и редактирование постов
- Добавление изображений
- Комментирование записей
- Профили пользователей
- Категории публикаций
- Пагинация

## Установка

1. Клонировать репозиторий:
```bash
git clone <url>
cd django_sprint4
```

2. Создать виртуальное окружение:
```bash
python3 -m venv venv
source venv/bin/activate
```

3. Установить зависимости:
```bash
pip install -r requirements.txt
```

4. Применить миграции:
```bash
cd blogicum
python manage.py migrate
```

5. Запустить сервер:
```bash
python manage.py runserver
```

Сайт доступен по адресу http://127.0.0.1:8000/

Админ-панель: http://127.0.0.1:8000/admin/
